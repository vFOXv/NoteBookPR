Приложение моделирует работу ежедневника.
При запуске по URL "/" открывается меню приложения
1 Show all notes- показывает все записи в дневнике
2 Date search on notes- поиск заметок по дате
3 Topic search on notes- поиск заметок по темам(возможно задавать более одной темы)
Заметка попадает в список найденных если в ее темах присутствует все заданные темы в поиске.
4 Create new note- создание новой заметки

5 Create new topic- создание новой темы
6 Delete topic- удаление темы из списка
(возможно, только если тема не используется ни в одной заметке)

Заметки и темы можно создавать только используя английскую раскладку.

Валидация при создании тем и заметок - нельзя создать тему или заметку в виде пустой строки в поле текст.

DB - код для создания и заполнения DB в папке resources/initDB.sql

При выборе пункта Show all notes -можно удалить и редактировать заметку
